=== FreeTranscriber Portable Kit ===

Installation & Run:
1. Unzip this folder anywhere.
2. Double-click "start.bat" (with console) OR "run_hidden.vbs" (silent mode).
3. The first run will automatically install required Python libraries.

Usage:
- Click the floating circle or press 'Ctrl+Shift+Space' to Record.
- Click again to Stop.
- Wait for the text to be typed automatically.
- Right-click the Tray Icon to Exit.

Requirements:
- Python 3.10+ installed and added to PATH.
- Internet connection (for first run to download model/libs).

GPU Support (Optional, for faster transcription):
- NVIDIA GPU with CUDA support
- CUDA Toolkit 11.x or 12.x installed
- cuDNN 8.x installed and configured
- Note: The application will automatically fall back to CPU if GPU is not available.
- For GPU setup instructions, visit: https://github.com/SYSTRAN/faster-whisper#gpu

Enjoy!
